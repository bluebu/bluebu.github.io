1、安装libjpeg

yum install -y libjpeg libjpeg-devel

2、下载源码

   http://freecode.com/projects/jpegoptim

   http://www.kokkonen.net/tjko/projects.html

3、解压

tar -zxvf jpegoptim-1.2.4.tar.gz

4、编译安装

cd jpegoptim-1.2.4
./configure
make && make install

ubuntu 就灰常简单了， 直接 apt-get install jpegoptim 就行了
