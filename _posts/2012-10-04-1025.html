从源代码中安装nginx

1、运行./configure报错：

./configure: error: the HTTP rewrite module requires the PCRE library.


2、安装 libpcre3 libpcre3-dev

apt-get install libpcre3 libpcre3-dev


3、重新编译

make && make install
